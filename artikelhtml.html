HTML

HTML merupakan singkatan dari "Hypertext Markup Language" atau "bahasa markup". Html adalah bahasa standar pemrogaman yang digunakan untuk membuat halaman website, yang diakses melalui internet.

ARTIKEL HTML

~ DASAR & SEJARAH HTML ~

Hypertext Markup Language atau HTML adalah bahasa markup standar yang digunakan untuk membuat halaman website dan aplikasi web. Sejarah HTML dimulai oleh Tim Berners-Lee, seorang ahli fisika di lembaga penelitian CERN yang berlokasi di Swiss. Versi pertamanya dirilis pada tahun 1991, dengan 18 tag. Sejak saat itu, setiap kali ada versi barunya, pasti akan selalu ada tag dan attribute (tag modifier) yang juga baru.

Ketika bekerja dengan bahasa markup ini , Anda menggunakan struktur kode sederhana (tag dan attribute) untuk mark up halaman website. Misalnya, Anda membuat sebuah paragraf dengan menempatkan enclosed text di antara tag pembuka <p> dan tag penutup </p>. Berkat popularitasnya yang terus meningkat, bahasa markup ini kini dianggap sebagai standar web resmi. Spesifikasi HTML dikelola dan dikembangkan oleh World Wide Web Consortium (W3C). Berdasarkan HTML Element Reference milik Mozilla Developer Network, saat ini ada 140 tag HTML, meskipun sebagiannya sudah tidak lagi didukung oleh beberapa versi terbaru browser.

Kemudian, pada tahun 2014, HTML5 mulai diperkenalkan. Dengan diperkenalkannya HTML5, terdapat semantic baru seperti <article>, <header>, dan <footer> yang memperjelas bagian-bagian dalam konten. Nah, setelah Anda tahu apa itu HTML, selanjutnya kita akan membahas cara kerjanya.

~ TAG HTML ~

Tag adalah tanda awalan dan akhiran dalam perintah HTML yang akan dibaca oleh web browser. Tag dibuat dengan menggunakan kurung siku <…>, di mana di dalamnya berisi nama tag. Setiap tag memiliki fungsi perintah yang berbeda-beda. Mulai dari membuat judul, paragraf, heading, cetak tebal, miring, italic, dan lainnya. Misalnya: <bold>.

Tag tersebut akan meminta browser untuk menampilkan teks dengan format tebal. Selain itu, tag ditulis secara berpasangan, yaitu tag pembuka dan tag penutup. Pada tag penutup ditambahkan garis miring (/) di depan nama tag. Berikut ini adalah tabel dari beberapa tag dasar yang ada di html.

Nama Tag Fungsi Tag
<html> Tag html berfungsi untuk memulai membuat halaman HTML yang mencakup semua konten dan elemen.
<head> Tag head merupakan tag yang membungkus bagian header dalam dokumen HTML. Tag ini berisi judul halaman web
<title> Title merupakan tag yang digunakan untuk membuat judul dari halaman. Teks yang diketikkan di dalam tag title akan ditampilkan di nama tab
<body> Body merupakan tag yang mewadahi isi dari konten web
<h1...h6> Tag h1 hingga h6 merupakan tag yang digunakan untuk menuliskan judul atau teks heading dari sebuah web
<p> Tag p digunakan untuk membuat sebuah paragraft dalam halaman web

~ ELEMEN HTML ~

Elemen merupakan komponen HTML yang berupa keseluruhan kode dari tag pembuka hingga tag penutup. Elemen terdiri dari teks dan simbol yang berupa tag pembuka, isi tag atau konten, dan tag penutup. Pada sebuah elemen juga bisa berisi elemen lainnya. Jadi elemen tersebut tak hanya berisi satu tag saja, tetapi banyak tag.



  1. 1. Elemen Heading

    2. Elemen heading merupakan elemen yang biasa digunakan untuk membuat judul. Elemen heading dibuah menggunakan tag <h> dengan nilai 1 hingga 6. Berikut ini adalah contoh penulisan elemen heading pada html

    Contoh Elemen Nama Elemen

    <head>
        <title>Elemen Heading</title>
    </head>
    <body>
        <h1> Heading 1 </h1>
        <h2> Heading 2 </h2>
    </body>

    		Elemen Head


  2. 2. Elemen Paragraf

    3. Elemen paragraf merupakan elemen yang digunakan untuk membuat sebuah paragraf pada html. Elemen paragraf terdiri terbentuk dari tag <p> dan isinya.

    Contoh Elemen Nama Elemen

    <head>
        <title>Elemen Paragraf</title>
    </head>
    <body>
        <p> Disini merupakan paragraf </p>
    </body>

    		Elemen Paragraf


  3. 3. Elemen Garis Horizontal

    4. Elemen ini digunakan untuk membentuk sebuah garis horizontal di dalam sebuah halaman web. Elemen ini dibuat menggunakan tag <hr> tanpa tag penutup

    Contoh Elemen Nama Elemen

    <head>
        <title>Elemen Garis Horizontal</title>
    </head>
    <body>
        <p> Disini merupakan paragraf </p>
        <hr>
        <p> Disini merupakan paragraf </p>
    </body>

    		Elemen Garis Horizontal


  4. 4. Elemen Baris Baru

    5. Elemen ini digunakan untuk memberi space atau baris baru pada halaman html. Untuk membuat elemen ini, digunakanlah tag <br>

    Contoh Elemen Nama Elemen

    <head>
        <title>Elemen Baris Baru</title>
    </head>
    <body>
        <p> Disini merupakan paragraf </p>
        <br>
        <p> Disini merupakan paragraf </p>
    </body>

    		Elemen Baris Baru


  5. 5. Elemen Komentar

    6. Elemen ini digunakan untuk mengkomentari sebuah blok kode, sehingga tidak ditampilkan di halaman web. Untuk membuat elemen ini, digunakanlah tag <!-- -->

    Contoh Elemen Nama Elemen

    <head>
        <title>Elemen Komentar</title>
    </head>
    <body>
        <p> Disini merupakan paragraf </p>
        <!-- Apapun yang ditulis disini adalah sebuah komentar -->
        <p> Disini merupakan paragraf </p>
    </body>

    		Elemen Komentar

~ ATRIBUT HTML ~

Atribut adalah informasi atau perintah tambahan yang berada dalam elemen. Atribut ini berfungsi sebagai penjelas perintah tag pada elemen. Misalnya, <img src=”gambar.jpg” alt “Bunga Matahari.”>. Tag <img> memiliki atribut khusus yaitu (scr) dan (alt) yang artinya browser harus menampilkan gambar.jpg dengan alt text “Bunga Matahari”.

Dari contoh di atas, dapat diketahui juga kalau jumlah atribut dalam sebuah tag bisa lebih dari satu. Meski demikian, tidak semua atribut bisa digunakan dalam sebuah tag. Terdapat 3 jenis atribut dalam html. Yang pertama adalah atribut global, kedua adalah atribut event, dan yang terakhir adalah atribut khusus.



  1. 1. Atribut Global

    2. Atribut Keterangan
    id Memberikan id unik pada sebuah elemen html
    class Memberikan sebuah class pada elemen html
    style Merupakan atribut yang digunakan untuk menyisipkan style khusus untuk elemen itu sendiri
    title digunakan sebagai judul informasi tambahan pada sebuah element.


  2. 2. Atribut Event

    3. Atribut Keterangan
    onclick Atribut yang dijalankan ketika elemen di klik
    onkeyup Atribut yang dijalankan ketika melepas tombol pada keyboard
    onkeydown Atribut yang dijalankan ketika menekan tombol pada keyboard
    onmouseover Atribut yang dijalankan ketika pointer berada di atas elemen


  3. 3. Atribut Khusus

    4. Atribut Keterangan
    href Atribut href berisi URL halaman website yang dibuka pada saat link diklik
    src Atribut src dipakai pada image (tag img). Atribut src berisi lokasi gambar atau URL gambar yang ditampilkan pada image
    width / height Atribut width dan height dipakai pada tag img. Atribut width dan height berisi ukuran gambar pada saat ditampilkan.
